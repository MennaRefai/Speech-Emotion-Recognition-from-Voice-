# Speech-Emotion-Recognition-from-Voice-
Built a speech emotion recognition model using CNN + LSTM architecture to detect emotions (happy, sad, angry, neutral, etc.) from audio files. Used Librosa to extract MFCCs, and mel-spectrograms from raw audio. Trained the CNN-LSTM model to capture both spatial and temporal features from the audio signals. Evaluated model accuracy on benchmark datasets
